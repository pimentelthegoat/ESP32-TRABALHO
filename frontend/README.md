# Frontend Vercel

Esta pasta e o site estatico que pode ser enviado para a Vercel.

## Como subir

1. Crie um projeto na Vercel.
2. Selecione esta pasta `frontend` como raiz do projeto.
3. Nao precisa de comando de build.
4. Depois do deploy, abra o site e confira o campo `Backend local`.

Por padrao o painel tenta conectar em:

```text
http://localhost:3000
```

Se seu backend estiver em outra porta ou IP, altere o campo na tela e clique em `Conectar`.
